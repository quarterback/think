# Think Studio

A really basic static HTML/CSS/JS portfolio for a solo artist working as a speculative designer. The site is inspired by simple editorial portfolio builders, uses Fontshare-hosted type instead of Google Fonts, and includes accessible structure, visible focus states, semantic landmarks, skip navigation, sufficient contrast, and reduced-motion support.

## Files

- `index.html` — semantic page content and navigation.
- `styles.css` — responsive styling and Fontshare imports.
- `script.js` — mobile navigation, current year, and progressive reveal effects.

## Deploy

This is a static site, so it can be deployed directly from GitHub to Vercel or Netlify.

### Vercel

1. Push this repository to GitHub.
2. Import the repo in Vercel.
3. Use the default settings. No build command is required.
4. Set the output directory to the project root if prompted.

### Netlify

1. Push this repository to GitHub.
2. Create a new Netlify site from Git.
3. Use the default static-site settings. No build command is required.
4. Set the publish directory to the project root if prompted.

## Customize

Replace `Think Studio`, `hello@example.com`, project titles, and project descriptions with your real artist name, contact details, and work.
